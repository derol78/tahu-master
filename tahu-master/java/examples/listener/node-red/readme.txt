RemoveLinesProject-1.0-SNAPSHOT.jar
- removs by default all lines that starts with "0". other values to be removed can be set as argument.
- exempel: java -jar RemoveLinesProject-1.0-SNAPSHOT.jar 3 - removs alla values of 3.

- input.csv - file to be processed
- output.csv - prcessed file
